### 📄 Crypto-Agent — From Hypotheses to Factors: Constrained LLM Agents in Cryptocurrency Markets

#### 1. 研究问题与方法（简述）

- **问题**：LLM Agent 若可随意改数据划分与评估规则，强结果可能来自 **未受控搜索** 而非有效发现。
- **方法**：**推理与评估分离**  
  - Agent（GPT-5.4）：读 append-only 实验 trace → 提出可证伪假设 → 映射为 **受限 DSL** 信号配方；  
  - **确定性评估引擎**：固定数据划分、门控阈值、成本、组合测试；Agent **不能** 在会话内修改评估规则。  
  - 五轮搜索后：good pool 因子 → **Ridge 回归** 合成 → 同一组合引擎 OOS 检验。

#### 2. 使用指标

| 指标 | 类别 | 依赖多空 |
|------|------|----------|
| P1 IC（日截面 Pearson） | 预测能力 | 否 |
| P6 IC t-stat | 稳健性/衰减 | 否 |
| T5 Sharpe（单因子 L-S 训练期） | 交易绩效 | 是 |
| T2/T4 AnnRet / Sharpe（组合） | 交易绩效 | 是 |
| T7 MaxDD | 交易绩效 / 稳健性 | 是 |
| T13 Calmar | 交易绩效 | 是 |
| T11 Turnover | 稳健性/衰减 | 是 |

**门控（仅训练期）**：mean IC 与 IC t-stat 超过配置阈值 → 进入 hold pool；验证期指标 **不用于** 改变门控。

#### 3. 多空组合实施细节

- **是否做空**：**是**。
- **单因子**：训练期报告 L-S Sharpe 等；**五分组 Quintile** 按预测信号排序（**C8**）。
- **组合信号**：Ridge 合成 score → 五分组；**L-S = Q4（long）− Q0（short）** 等权。
- **权重版本**：等权（EW）与市值加权（VW）；VW 表现差 → alpha 集中在 **小市值**，容量受限。
- **成本**：净收益分析；主结果 **单边 5 bp（0.05%）**；敏感性 0.1%–0.3%（Table 3）。
- **换仓**：日频滚动窗口框架内组合再平衡（quintile 排序）。

#### 4. 实验环境

| 项目 | 设置 |
|------|------|
| 数据 | CoinMarketCap 日频，2020-01–2025-12；过滤上市 <180 日、低流动性 |
| 划分 | **训练 2020–2022**（选因子 / 门控 / Ridge 拟合）→ **验证 2023** → **纯 OOS 2024+** |
| 搜索 | 5 轮；机械候选（lag/MA/zscore）+ Agent 假设候选 |
| 机制结论 | 收敛至 **小市值 + 高日内 range + 低流动性/注意力** 的 trend-continuation 族 |
| 主结果（Combo EW L-S） | 纯 OOS：**年化 44.55%**，Sharpe **1.55**，MaxDD −23.6%，Turnover 0.368（5bp 成本后） |
| 单因子 | 25 个因子；最佳纯 OOS Sharpe 约 **2.41**（smallcap lowvol 等） |

#### 5. 评估文档交叉引用

- 通用 IC / Sharpe / MDD：上级 §一 P1、T5、T7  
- 本论文特有：P6 门控 t-stat、C8 五分组、固定协议不可由 Agent 修改

### 📄 Crypto-GPT — LLMs and Generative Factor Discovery in Crypto Markets (SSRN)

> SSRN 预印本，**未经同行评审**。

#### 1. 研究问题与方法（简述）

- **问题**：加密市场因子多靠手工启发式，数据挖掘与 OOS 可靠性存疑；传统 ML 限于预设特征空间。
- **方法**：**GPT-5.2 代码生成因子**（BitMEX USD 永续 + CoinMarketCap 市值/成交量）  
  - **Zero-shot**：基金管理者 persona + 数据字典，顺序生成 30 个 Python 因子；  
  - **三阶段筛选**：静态前瞻偏差检测 → 沙箱执行 → 统计有效性（方差/缺失）；  
  - **Few-shot**：将优因子代码与绩效反馈注入 prompt，再生成 50 个因子；  
  - 跨模型：Gemini 3 Pro、Grok 4 Expert；跨版本：GPT-4o。

#### 2. 使用指标

| 指标 | 类别 | 依赖多空 |
|------|------|----------|
| P2 Rank IC | 预测能力 | 否 |
| P3 ICIR | 稳健性/衰减 | 否 |
| P2 t-stat（Rank IC） | 预测能力 / 稳健性 | 否 |
| T2/T3 Ann Ret（L/S 超额） | 交易绩效 | 是 |
| T5 Sharpe | 交易绩效 | 是 |
| T7 MaxDD | 交易绩效 / 稳健性 | 是 |
| P7 Alpha（回归） | 交易绩效 | 是 |
| R4 训练截止 OOS | 稳健性/衰减 | 是 |

#### 3. 多空组合实施细节

- **是否做空**：**是**（主策略为 L/S）；另报告 **仅做多 Top 20%**（L/O）。
- **范式**：**C7** — L/S 买 Top 20%、卖 Bottom 20%；L/O 仅持有 Top 20%。
- **信号方向**：**60 日滚动 mean RankIC**；若为负则 **翻转持仓符号**（反转类因子）。
- **收益计算**：在 **现货** 上计算策略收益（用衍生品数据预测现货趋势，规避合约杠杆噪声）。
- **成本**：**单边 0.05%** 日权重调整成本。
- **多因子组合**：
  - **静态等权** 标准化信号；
  - **动态自适应**：按近期 L/S（或 L/O）绩效筛选因子加权（Panel B 动态 L/S 年化 373.36%，Sharpe 3.21，MDD −44.98%）。

#### 4. 实验环境

| 项目 | 设置 |
|------|------|
| 标的 | BitMEX 全上市 USD 永续；CoinMarketCap 市值/量 |
| 样本 | 全历史至 **2025-12-31**（in-sample 统计与回测） |
| **严格 OOS** | GPT-5.2 训练截止 **2025-08-31** → OOS **2025-09-01–2025-12-31**（**R4**） |
| Zero-shot | 30 因子；最佳 Factor 22：RankIC −0.10，t=−12.12 |
| In-sample L/S 顶因子 | Factor 22：**658.01%** 年化超额，Sharpe **4.13**，MDD −27.78% |
| OOS Factor 22 | **807.71%** 年化，Sharpe **4.63**，MDD **−6.55%** |
| Few-shot | 提升部分尾部因子，但增加复杂性与过拟合（Sharpe 可为负） |
| GPT-4o 对照 | 最佳因子年化约 99.12%，远低于 GPT-5.2 |

#### 5. 评估文档交叉引用

- Rank IC / ICIR：上级 P2、P3  
- L/S Top20%：本文件夹 **C7**（与主目录 C2/C3 的 Top 比例不同）  
- 结果数值极高，解读需考虑 **加密高波动、小样本 OOS（4 个月）** 与 **未同行评审**
